What is the Quotz API?

The Quotz API is a small service designed to provide motivational and inspirational quotes based on your preferences. Perfect for apps, websites, and personal projects!

How to Use the API

Make a GET request to the API endpoint with the following parameters to retrieve quotes:

Parameter Description Required
api Your unique API key. Register to get your key. Yes
qn Number of quotes to retrieve (e.g., 1, 5, 10). Optional (default: 1)
qs Quote size: small, medium, or long. Optional (default: small)
tag Filter quotes by a specific tag (e.g., inspirational, life). Optional

Example API Calls

Retrieve 3 short quotes about life:

GET /quotes?api=your_api_key&qn=3&qs=small&tag=life

Retrieve a single random medium-sized quote:

GET /quotes/random?api=your_api_key&qs=medium
